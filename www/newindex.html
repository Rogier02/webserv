[webserv]
01

POST — upload file

Upload a file to the server. The server stores it under /uploads/ via the CGI handler.

02

GET — browse uploads

Open the /uploads/ directory listing served by the server.

Open /uploads/
03

DELETE — remove a file

Browsers cannot send a DELETE request natively, so this routes through a CGI helper (delete.py) that receives the target path as a POST field and performs the deletion server-side.

04

GET — CGI environment

Opens /cgi-bin/env.py which dumps every environment variable the server passes to the CGI process. Varifies REQUEST_METHOD, CONTENT_TYPE, PATH_INFO, QUERY_STRING, etc.

Open env dump
05

HEAD — use curl

Browsers never expose HEAD responses to the page, so test it from the terminal: 

curl -I http://localhost:8080/
curl -I http://localhost:8080/uploads/
curl -I http://localhost:8080/cgi-bin/env.py
